Getting Started

Dependencies

One of the goals for Sam Design System Components is to create Angular components that do not depend on the USWDS 2.0 native javascript libraries. However, there are a few peer dependencies that are required depending on the package(s) you choose to import.

Packages Available
Name Description

@gsa-sam/components
Beta.SAM.gov common Angular components (e.g. accordion, modal dialog, tables, etc)
Depends on
@gsa-sam/sam-styles
@fortawesome/angular-fontawesome@^0.3.0
@fortawesome/fontawesome-svg-core@^1.2.17
@fortawesome/free-solid-svg-icons@^5.8.1

@gsa-sam/layouts
Beta.SAM.gov components that build the structure of the site (e.g. header, footer, sidenav, etc)
Depends on
@gsa-sam/sam-styles

@gsa-sam/sam-formly
Beta.SAM.gov form elements based on ngx-formly (e.g. input, select, date picker, etc)
Depends on
@gsa-sam/sam-styles
@gsa-sam/components@^0.6.0
@ngx-formly/core@5.4.1
@angular/material@7.3.7
@fortawesome/angular-fontawesome@^0.3.0
@fortawesome/fontawesome-svg-core@^1.2.17
@fortawesome/free-solid-svg-icons@^5.8.1

Supported Browsers

Our components support the latest versions of Chrome, Firefox and Internet Explorer 11.
